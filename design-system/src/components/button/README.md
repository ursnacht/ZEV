# Button Component

Wiederverwendbare Button-Komponente mit verschiedenen Varianten.

## Verwendung

```html
<!-- Primary Button -->
<button class="zev-button zev-button--primary">Click Me</button>

<!-- Secondary Button -->
<button class="zev-button zev-button--secondary">Secondary</button>

<!-- Danger Button -->
<button class="zev-button zev-button--danger">Delete</button>

<!-- Disabled -->
<button class="zev-button zev-button--primary" disabled>Disabled</button>

<!-- Compact -->
<button class="zev-button zev-button--primary zev-button--compact">Compact</button>
```

## Varianten

- `zev-button--primary`: Primäre Aktion (Grün)
- `zev-button--secondary`: Sekundäre Aktion (Blau)
- `zev-button--danger`: Destruktive Aktion (Rot)
- `zev-button--compact`: Kompakte Variante mit weniger Padding

## States

- `:hover`: Dunklere Hintergrundfarbe
- `:disabled`: Grauer Hintergrund, nicht klickbar
